⚙️ Installation Guide

1️⃣ Clone Repository

git clone https://github.com/your-username/rbajee-token-platform.git
cd rbajee-token-platform

2️⃣ Setup Server

cd server
# install dependencies
# start server

3️⃣ Setup Database

"database" ফোল্ডারের ".sql" ফাইল MySQL বা PostgreSQL database-এ import করুন।

4️⃣ Run Admin Panel

"admin-panel" ফোল্ডার web server (Apache / Nginx / Node) দিয়ে run করুন।

5️⃣ Build Android App

cd android-app
# build APK
